## What is DSA?
DSA stands for **Data Structures and Algorithms**.  
- **Data Structures** are ways to store and organize data in a computer.  
- **Algorithms** are step-by-step instructions to solve problems using that data.  

Learning DSA helps write code that is **efficient, fast, and scalable**.

---

## What is Time Complexity?
Time Complexity tells us **how long a program takes to run** depending on the size of the input.  
It helps us choose **faster and better solutions**.

### Why is it important?
- Two programs can solve the same problem differently.  
- One may take 1 second, another may take 1 hour.  
- Time complexity gives a mathematical way to compare efficiency.

---

## Common Time Complexities and Simple Examples

### O(1) – Constant Time
- **Meaning:** Time does not change with input size.  
- **Example:** Picking the first book from a shelf.  

### O(log n) – Logarithmic Time
- **Meaning:** Each step reduces the problem size roughly by half. Very fast.  
- **Example:** Searching a name in a phone book by opening the middle, deciding left or right, and repeating.  

### O(n) – Linear Time
- **Meaning:** Work grows in direct proportion to input size.  
- **Example:** Checking each item in a shopping list one by one.  

### O(n²) – Quadratic Time
- **Meaning:** Work grows like n × n.  
- **Example:** Every student shakes hands with every other student.  

### O(n³) – Cubic Time
- **Meaning:** Work grows with three nested loops (n × n × n).  
- **Example:** Checking all possible triplets in a list of people.  

### O(2ⁿ) – Exponential Time
- **Meaning:** Work doubles with every new input.  
- **Example:** Testing all ON/OFF combinations of n switches.  

### O(n!) – Factorial Time
- **Meaning:** Work checks every possible arrangement. Extremely slow.  
- **Example:** Arranging n people in all possible orders for a photo.

---

## Key Takeaways
- Start with simple, fast approaches (O(1), O(log n), O(n)).  
- Avoid exponential or factorial solutions unless input size is tiny.  
- Real-life examples help understand concepts better.
